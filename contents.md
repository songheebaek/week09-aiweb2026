## 🧩 어떤 문제를 풀까요?

> YouTube엔 강의·뉴스·인터뷰·기술 영상이 넘쳐나지만, **영상이 길어 다 보기 어렵습니다.**  
> 필요한 정보 한 줄을 찾으려 처음부터 끝까지 돌려봐야 할 때도 많죠.

특히 **학습용 영상**은 핵심을 빠르게 파악하고, 원하는 대목을 다시 찾는 기능이 필요합니다.  
그래서 AI로 자막을 분석해 **핵심 정보만을 효율적으로** 전달하는 서비스를 만들었습니다.

- 💼 인터뷰·뉴스를 빠르게 훑는 **직장인**
- ⏱️ 긴 영상의 핵심만 먼저 확인하고 싶은 **모든 사용자**

## 🗺️ 전체 동작 흐름

1. **🔗 URL 입력** — 영상 주소를 넣고 `[요약하기]` 클릭
2. **🔍 영상 분석** — URL에서 영상 ID 추출
3. **📜 자막 추출** — 한국어 우선, 없으면 영어
   - 🖥️ **로컬 실행** — 개인 IP로 YouTube에 직접 접근해 자막을 가져옵니다.
   - ☁️ **클라우드 배포** — 서버(데이터센터) IP는 YouTube가 자동화로 간주해 **차단**하는 경우가 많습니다.
     → 그래서 **Webshare 프록시**를 경유해, 일반 사용자 IP처럼 보이게 함으로써 배포 환경에서도 **자막 추출이 안정적으로 동작**하도록 합니다.
4. **🧹 전처리** — `[분:초] 내용` 타임스탬프 형태로 변환
5. **🤖 AI 분석** — 단일 호출로 요약 + 타임스탬프 핵심 동시 추출
6. **📊 결과 출력** — 요약 · 타임스탬프별 핵심 · Q&A
7. **💬 영상 Q&A (선택)** — 질문 입력 시 자막을 근거로 답변
