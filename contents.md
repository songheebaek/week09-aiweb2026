# 나의 스트레스 몬스터

사용자가 오늘의 스트레스를 한 문장 이상 입력하면 AI가 스트레스 유형을 분석해 귀엽고 웃긴 스트레스 몬스터 캐릭터 카드를 생성하는 서비스

- [ ] C. 외부 API + LLM (공공/외부 API + LLM 가공)
- [ ] 자유 (강사 사전 협의 필요)

## 사용자

[누가 쓰는가 — 본인 / 특정 그룹 / 일반 대중 중 선택. 구체적인 인물상 1~2문장]

---

## 핵심 기능 (3개 이하)

1. 사용자의 스트레스 문장을 분석하여 스트레스 유형 판단
2. 스트레스 특징을 반영한 몬스터 캐릭터 이미지 생성
3. 스트레스 지수, 몬스터 이름, 특징 등을 포함한 “오늘의 스트레스 몬스터 카드” 생성

---

## 입력 → AI 처리 → 출력

사용자 스트레스 텍스트 입력 → GPT API로 스트레스 유형 및 특징 분석 → 이미지 생성 AI로 몬스터 캐릭터 생성 → 스트레스 지수 및 설명이 포함된 몬스터 카드 출력

---

## 기술 스택

- **UI**: Streamlit
- **모델 / API**: GPT-4o-mini, OpenAI Image Generation API
- **외부 서비스**: 없음

---

## 데이터 출처

사용자가 직접 입력한 스트레스 문장을 활용하며, 별도의 외부 데이터셋은 사용하지 않는다.

⚠️ 저작권·개인정보 위반 데이터는 ✗

---

## 완료 조건

사용자가 최소 한 문장 이상의 스트레스 내용을 입력하면 30초 이내에 스트레스 유형 분석 결과와 함께 몬스터 캐릭터 이미지 및 “오늘의 스트레스 몬스터 카드”가 생성된다.
