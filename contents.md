# AI YouTube Summarizer - YouTube 영상 요약기

사용자가 YouTube 영상 URL을 입력하면 자막을 자동으로 추출하고 AI가 핵심 내용을 요약하여 제공하는 서비스. 긴 영상을 모두 시청하지 않아도 주요 내용을 빠르게 파악할 수 있으며, 타임스탬프별 핵심 내용과 영상에 대한 질의응답 기능까지 제공한다.

---

## 트랙

외부 API + LLM (영상 자막을 외부 API로 가져옴 → LLM(Gemini)으로 가공(요약))

---

## 인프라 트랙

HF Spaces

---

## 문제

YouTube에는 유익한 강의, 뉴스, 인터뷰, 기술 영상 등 다양한 콘텐츠가 존재하지만 영상 길이가 길어 모든 내용을 시청하기 어렵다. 필요한 정보를 찾기 위해 영상을 처음부터 끝까지 확인해야 하는 경우도 많다. 특히 학습 목적의 영상은 핵심 내용을 빠르게 파악하고 특정 내용을 다시 찾을 수 있는 기능이 필요하다. 따라서 AI를 활용하여 영상 자막을 분석하고 핵심 정보를 효율적으로 제공하는 서비스를 만들고자 한다.

---

## 사용자

학습용 강의, 기술 영상, 인터뷰, 뉴스 등을 자주 시청하는 학생 및 직장인. 긴 영상을 모두 시청하기 전에 핵심 내용을 빠르게 파악하고 싶은 일반 사용자.

---
## 핵심 기능

1. YouTube URL 입력 시 영상 자막 자동 추출
2. AI를 활용한 영상의 전체 핵심 내용 요약
3. 타임스탬프별 핵심 링크 제공 및 영상 내용 Q&A

---

## 입력 → AI 처리 → 출력

URL 입력 → youtube-transcript-api로 자막 추출 → Gemini 2.5 Flash가 자막 분석 및 전체 요약, 타임스탬프 생성 → 전체 요약/타임스탬프별 핵심 내용/Q&A 출력

## 데이터 출처

YouTube 영상에서 제공되는 자막 데이터를 활용한다. 사용자가 입력한 영상 URL에 대해서만 자막을 추출하며, 별도의 데이터셋을 저장하거나 수집하지 않는다.
